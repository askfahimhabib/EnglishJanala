## 📖 About The Project

**EnglishJanala** (English Window) is a front-end web application built to make learning English more accessible and engaging. Whether it's vocabulary, grammar, or reading practice, this project aims to provide a clean and interactive learning environment.

## ✨ Features

- 🌍 **Interactive UI:** A clean, easy-to-navigate, and responsive user interface.
- 📱 **Mobile Friendly:** Designed to work smoothly across desktops, tablets, and mobile devices.
- ⚡ **Fast & Lightweight:** Built using core web technologies for maximum performance.

## 💻 Tech Stack

- **HTML5** - For the structure of the application.
- **CSS3** - For styling, layouts, and animations.
- **JavaScript** - For adding interactivity and logic.
